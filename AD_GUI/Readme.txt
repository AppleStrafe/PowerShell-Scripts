#######################################################

Dependiancies:
= Install-Module MSOnline
- Install-Module AzureADPreview

#######################################################

Services this script offers:
- Onboarding
- User tansfers
- User rehire
- Terminations

#######################################################

How to use:

Install all the modules:
1) Run the Setup.ps1
2) Enter admin credentials
3) Type "A" and click enter

Run the main script:
1) Right click on the MainWindow.ps1
2) Select run with powershell
3) Enter admin credentials
4) Enter admin credentials for azure
5) Select what you would like to do on the left side of the GUI
